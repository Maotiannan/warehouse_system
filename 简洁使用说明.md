# 仓库查询系统使用说明

## 启动方式

当前仓库只保留一种脚本启动方式：

- 双击 `简洁启动.bat`
- 实际执行命令：
  `javaw -jar target/WarehouseQuerySystem-1.0-SNAPSHOT.jar`

如果使用安装包，则直接通过开始菜单或桌面快捷方式启动安装后的程序，不需要再额外运行其他 bat。

## 当前主要功能

- 仓库数据查询
- 高级筛选
- 查询日志查看
- 数据导出
- 列配置保存
- 唛头本地编辑和本地缓存覆盖

## 唛头本地编辑

- 查询结果仍然每次联网获取
- 唛头列支持直接编辑并本地保存
- 同一 `zyh` 后续再次查询时，优先显示本地保存过的唛头
- 本地缓存文件位置：
  `C:\Users\<当前用户>\.warehouse-query-system\entry-cache.json`

## 手动检查重点

1. 启动后确认右侧结束日期自动是今天
2. 查询库存编号 `L25MH070359`，确认货主不为空
3. 查询 `L25MH121026`，点击“照片”和“托信息”，确认 URL 中 `jcid=` 后使用的是 `inguid`
4. 修改任意记录唛头后重新查询同一 `zyh`，确认优先显示本地保存值

## 生成产物

- fat jar：
  `target/WarehouseQuerySystem-1.0-SNAPSHOT.jar`
- Windows 安装包本地构建输出：
  `target/installer/WarehouseQuerySystem-1.0.1.exe`
- GitHub 最新安装包请从 Releases 下载
