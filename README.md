# 仓库查询系统

基于Java + JavaFX的仓库查询系统桌面应用程序，支持多种查询状态和完整的数据管理功能。

## 🚀 快速开始

### 第一次使用
1. **创建桌面快捷方式**：双击 `create_shortcut.bat`
2. **移动快捷方式**：将生成的 `仓库查询系统.lnk` 剪切到桌面：`D:\文档\桌面`
3. **启动应用**：双击桌面快捷方式

### 启动方式
| 启动方式 | 窗口状态 | 推荐度 |
|---------|---------|--------|
| 桌面快捷方式 | 无窗口 | ⭐⭐⭐⭐⭐ |
| `启动仓库查询系统.bat` | 无窗口 | ⭐⭐⭐⭐ |
| `调试启动.bat` | 显示详情 | 🔧 调试用 |

## 💡 主要功能

### 数据查询
- **多状态查询**：预约进仓、进仓、库存、出仓
- **批量查询**：支持多个进仓编号（逗号分隔）
- **日期范围**：灵活的时间区间设置
- **实时数据**：直连API获取最新数据

### 数据管理
- **30字段完整支持**：包含所有业务字段
- **列配置管理**：显示/隐藏、宽度调整、个人配置保存
- **数据导出**：Excel格式导出选中数据
- **统计信息**：自动计算汇总数据

### 用户体验
- **启动优化**：使用javaw命令完全无命令行窗口
- **配置保存**：个人设置自动保存和恢复
- **错误处理**：完善的异常处理和用户提示
- **日志记录**：详细的操作和错误日志

## 🔧 技术栈

- **Java 11+** - 核心开发语言
- **JavaFX 11** - 桌面UI框架
- **Maven** - 项目管理和构建
- **Apache HttpClient** - HTTP请求处理
- **Jsoup** - HTML解析
- **Apache POI** - Excel文件处理

## 📋 系统要求

- Java 11 或更高版本
- Windows 10/11
- 网络连接（访问API服务器）

## 🆘 故障排除

### 启动问题
1. 双击 `调试启动.bat` 查看详细错误
2. 检查Java环境：`java -version`
3. 确保文件完整：lib目录和jar文件存在

### 查询问题
1. 检查网络连接
2. 确认进仓编号格式正确
3. 查看日志文件：`request_log.txt` 和 `response_log.txt`

## 📊 数据字段

支持完整的30个业务字段：
- **基础信息**：jcid, zyh, jcrq, jcbh, hz, mt
- **预计信息**：yjjs, yjtj, yjmz, yjrq
- **实际信息**：js, tj, mz
- **库存信息**：kcjs, kctj, kcmz
- **货物信息**：bzgg, hwmc, hwmc1
- **时间信息**：xhrq, xhrq2
- **物流信息**：ch, driverdh, cleng, chengzhong
- **其他信息**：lf, yyh, bz2, hdmc, fgsmc
