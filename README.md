# 仓库查询系统

基于 Java + JavaFX 的桌面应用，用于仓库数据的查询与管理，支持多状态查询、批量查询与数据导出。

**主要功能**
- 多状态查询：预录、进仓、库存、出仓等状态
- 批量查询：支持多个进仓编号（逗号分隔）
- 日期范围筛选
- 数据导出：Excel 格式
- 列显示配置：显示/隐藏、宽度调整、个人配置保留
- 操作与错误日志记录

**运行环境**
- Java 11 或更高版本
- Windows 10/11
- 可访问 API 的网络连接

**快速开始**
- 推荐启动：双击 `启动仓库查询系统.bat`
- 简洁启动：双击 `简洁启动.bat`
- 诊断启动：双击 `诊断启动差异.bat` 以查看启动差异与日志

**快捷方式相关**
- 创建快捷方式：运行 `直接创建快捷方式.bat`
- 修复快捷方式：运行 `修复快捷方式.bat`

**开发与构建**
- 构建命令：`mvn -DskipTests package`
- 构建产物：`target/`

**项目结构**
- `src/main/java/` Java 源码
- `src/main/resources/fxml/` JavaFX 界面文件
- `lib/` 本地依赖或运行时库
- `target/` 构建输出
- `dist/` 打包输出
- `temp_jpackage/` 打包临时目录
- `logs/` 运行日志

**日志与排错**
- 启动异常：优先运行 `诊断启动差异.bat`
- 请求日志：`request_log.txt`
- 响应日志：`response_log.txt`
- 调试输出：`session_response.html`

**许可**
本项目仅供内部使用。
